# Fonts Setup - خطوط Zekr Alrahman

## الخطوط المطلوبة

انسخ ملفات الخطوط التالية إلى `assets/fonts/`:

### Cairo (عناوين وأزرار)
- Cairo-Regular.ttf
- Cairo-Bold.ttf
- المصدر: Google Fonts

### Amiri (نصوص الأذكار)
- Amiri-Regular.ttf
- Amiri-Bold.ttf
- المصدر: Google Fonts

### Aref Ruqaa (أسماء الله)
- ArefRuqaa-Regular.ttf
- المصدر: Google Fonts

### Reem Kufi (عناوين فرعية)
- ReemKufi-Regular.ttf
- المصدر: Google Fonts

### Almarai (نصوص إرشادية)
- Almarai-Regular.ttf
- المصدر: Google Fonts

### Tajawal (نصوص بديلة)
- Tajawal-Regular.ttf
- المصدر: Google Fonts

## التحميل

```bash
# عبر Expo
npx expo install expo-font

# ملاحظة: الخطوط مُدرجة في app.json plugins
# Expo يقوم بتحميلها تلقائياً عند البناء
```
